# Against the Apocalypse

Source: `Mage-Knight-The-Apocalypse-Dragon---Rulebook-WEB.pdf`, pages 26–28.

A short scenario centered on possessed enemies and destroyed/restorable sites. Not a v1 target (v1 is Solo Conquest only) — captured here as reference for when this scenario gets implemented.

## Overview

- Length: Three rounds (2 days and 1 night).
- Purpose: creatures of the Apocalypse are spewing from a magical pyramid, destroying settlements. Stop the terror and restore the settlements.
- Uses possessed enemies and Faction tokens (see the expansion's "New Rules" section for possessed-enemy mechanics generally — out of scope here).

## Setup (all variants)

Place the Against the Apocalypse scenario card nearby. Roll a mana die and place a neutral Shield token on the card, covering the mana symbol of the color rolled.

### Special rules

- Whenever a new Map tile is revealed, check the neutral Shield token's row on the scenario card:
  - If it's in a row matching a site type present on the new tile, that site is destroyed (place a Destroyed Site token on it). Restoring a destroyed monastery does not add an Advanced Action to the Unit offer.
  - If it's in the Rampaging Enemies row and a rampaging enemy is on the new tile, that enemy (the leftmost, if several) becomes an **Apocalypse possessed enemy** — reveal a possessed-enemy token underneath its normal token.
  - Either way, move the neutral Shield token down one row (wrapping to the top row after the bottom).
  - If the indicated site/enemy type isn't actually present, keep re-checking at the new row until a match is found — every newly revealed tile ends up with exactly one destroyed site or one possessed rampaging enemy, except tiles revealed after the Map Tile deck has run out (filler tiles are exempt).
- Destroyed sites can be **restored**: the scenario card lists what's required per site type (usually being on the site). Restoring returns the Destroyed Site token to your Inventory and grants the reward shown on the scenario card (restoring a Mage Tower does *not* grant a Spell).
- Defeating an apocalypse-possessed **green** rampaging enemy → gain 1 Destroyed Site token into your Inventory. Defeating a possessed **red** rampaging enemy → gain 2.
- Optional Blitz variant: start with 1 Fame and 2(+1) Reputation, +1 Fame per level gained, one extra Source die and one extra Unit in the offer.

### Competitive

- Players: 2 to 4.
- Map Shape: Wedge, Wedge, or Open limited to 4 columns.
- Countryside tiles: 5, 6, or 8 (always include both ziggurat countryside tiles).
- Core city tiles: 1. Core non-city tiles: 1, 2, or 3 (always include exactly 1 tile with a pyramid).
- City: friendly — each player puts one Shield token on it, no-one is its leader.

### Cooperative

- Players: 2 to 4.
- Map Shape: Fully open.
- Countryside tiles: 5, 7, or 9 (always include both ziggurat tiles).
- Core city tiles: 1. Core non-city tiles: 2, 3, or 4 (always include exactly 1 pyramid tile).
- City: friendly, same as Competitive.
- Dummy Player: one standard Dummy Player. Cards/Skills: remove the four competitive Spells and the one competitive interactive Skill from each player's Skill tokens; use cooperative interactive Skill tokens if available.
- Tactics: Dummy Player takes a random Tactic first, then the players choose theirs; at the end of the first Day, agree and remove one player-used Tactic from the game. Team Rules apply, everyone is one team.

### Solo

- Players: 1.
- Map Shape: Wedge.
- Countryside tiles: 4 (always include exactly 1 ziggurat tile).
- Core city tiles: 1. Core non-city tiles: 1 (with a pyramid).
- City: friendly — both the player and the Dummy Player put a Shield token on it, no-one is its leader.
- Dummy Player: one standard Dummy Player. Cards/Skills: same removal/substitution as Cooperative.
- Tactics: player always chooses first; Dummy Player takes one random remaining card. At the end of each Day/Night, remove both used Tactic cards from the game.

## Scenario end

- Competitive/Cooperative: triggered once (a) total Destroyed Site tokens across all players' Inventories ≥ players + 1 (Competitive) or players + 2 (Cooperative), (b) conquered ziggurat+pyramid floors ≥ the same threshold, and (c) at least one floor of both ziggurats and the pyramid has been conquered. Then all players (Competitive) / all players except the Dummy Player (Cooperative) get one last turn.
- Solo: triggered once you have ≥2 Destroyed Site tokens in your Inventory *and* one floor of both the ziggurat and pyramid have been conquered. You get one last turn (the Dummy Player does not).
- In all variants, the Round ends immediately if the Round limit is hit during the last-turn sequence.

## Scoring

### Competitive

Success = all end conditions met. Apply standard Achievements Scoring, plus:
- +3 Fame per Destroyed Site token in a player's Inventory; highest → +5 Fame Greatest Restorer (+2 if tied).
- Fame per conquered ziggurat/pyramid floor: ziggurat floor 1/2/3 → 1/2/3 Fame; pyramid floor *n* → 2*n* Fame (these also count toward Greatest Adventurer per `solo-scoring-overview.md`). Highest total → +5 Fame Greatest Ascender (+2 if tied).

### Cooperative

One team score = lowest Fame among all players as base, then standard Achievements Scoring except only the highest-scoring player counts per category (no Titles). Plus:
- +3 points per Destroyed Site token in any Inventory.
- +5 points per conquered ziggurat/pyramid floor.
- +15 points if victorious.
- +30 points per Round finished before the Round limit.
- +1 point per unflipped card in the Dummy Player's Deed deck.
- +5 points if "End of the Round" wasn't announced yet in your last Round.

### Solo

Base = your Fame. Standard Achievements Scoring, no Titles. Plus:
- +3 points per Destroyed Site token in your Inventory.
- +5 points per ziggurat/pyramid *site* you've conquered a floor of (flat per site, 0-10 total: 5 for the ziggurat + 5 for the pyramid; unlike Competitive, the floor number reached - 1, 2, or 3 - doesn't change the score).
- +15 points if victorious.
- +30 points per Round finished before the Round limit.
- +1 point per unflipped card in the Dummy Player's Deed deck.
- +5 points if "End of the Round" wasn't announced yet in your last Round.

## Outcome (Won/Lost)

Won iff all scenario-end conditions were met (all Destroyed Site/floor thresholds reached); Lost otherwise. A score is always computed either way.
