# Apocalypse is Here

Source: `Mage-Knight-The-Apocalypse-Dragon---Rulebook-WEB.pdf`, pages 34–38.

The expansion's full campaign-style scenario: stop the Four Horsemen (which empower the Dragon's heads as they destroy sites) and then defeat the Apocalypse Dragon itself. Not a v1 target (v1 is Solo Conquest only) — captured here as reference for when this scenario gets implemented.

## Overview

- Length: Six rounds (3 days and 3 nights).
- Purpose: stop the Four Horsemen and defeat the Apocalypse Dragon before it's too late.

Key facts relevant to scoring/setup only:
- The Horsemen always take the last turn each Round (before the Dragon exists on the map).
- The Apocalypse Dragon is placed when the second City tile is revealed (covering the city space and 2 spaces above it); any rampaging enemies on that City tile become Apocalypse-possessed.
- If a Horseman destroys 4 sites, it's removed from the game (job done) without being "defeated." Any Horsemen still on the map when the Dragon is first attacked are removed without counting as defeated.
- Combat against a Horseman or the Dragon uses the rulebook's dedicated combat rules (referenced, not reproduced here).
- Optional "Horsemen's Horses" variant: pair each Horseman token with a face-up brown enemy token that moves with it and must be fought alongside it; defeating the horse scores its normal Fame, but a horse left undefeated when its Horseman is beaten (or removed) is discarded for no Fame.

## Setup

### Competitive

- Players: 2 to 4.
- Map Shape: Wedge, Wedge, or Open limited to 4 columns.
- Countryside tiles: 7, 8, or 10. Core city tiles: 2. Core non-city tiles: 1, 2, or 3.
- Cities: the first revealed is friendly (each player puts a Shield token on it, no-one leads it); the second revealed has already been destroyed by the Dragon — treat it as a Plains space with no site.
- Horsemen start at level 5. Each Dragon head's starting level = number of players + 2.

### Cooperative

- Players: 2 to 4.
- Map Shape: Fully open.
- Countryside tiles: 8, 9, or 11. Core city tiles: 2. Core non-city tiles: 2, 3, or 4.
- Cities: same first-friendly/second-destroyed setup as Competitive.
- Horsemen start at level 6. Each Dragon head's starting level = number of players + 4.
- Dummy Player: one standard Dummy Player. Cards/Skills: remove the four competitive Spells and the one competitive interactive Skill from each player's Skill tokens; use cooperative interactive Skill tokens if available.
- Tactics: Dummy Player takes a random Tactic first, then the players choose theirs. Team Rules apply, everyone is one team.

### Solo

- Players: 1.
- Map Shape: Wedge.
- Countryside tiles: 7. Core city tiles: 2. Core non-city tiles: 2.
- Cities: first revealed is friendly (both the player and the Dummy Player put a Shield token on it, no-one leads it); second revealed already destroyed — treat as a Plains space with no site.
- Horsemen start at level 4. Each Dragon head starts at level 5.
- Dummy Player: one standard Dummy Player. Cards/Skills: same removal/substitution as Cooperative.
- Tactics: player always chooses first; Dummy Player takes one random remaining card. At the end of each Day/Night, remove both used Tactic cards from the game (each Tactic card is picked exactly once).

## Scenario end

When the Apocalypse Dragon is defeated: all players (Competitive, including whoever defeated it) or all players except the Dummy Player (Cooperative) get one last turn; Solo: the player gets one last turn (the Dummy Player does not). The Round ends immediately if the Round limit is hit mid-sequence.

## Scoring

### Competitive

Success = Dragon defeated. Standard Achievements Scoring, plus for each head except the Control head:
- Each player scores Fame equal to their count of Shield tokens on that head's large enemy token; if they also defeated the corresponding Horseman, add that Horseman's level to the count.
- Highest total → +5 Fame Greatest Slayer of that head (ties broken toward whoever's Shield token sits on the highest level).

### Cooperative

One team score = lowest Fame among all players as base, then standard Achievements Scoring except only the highest-scoring player counts per category (no Titles). Plus:
- +3 points per Horseman defeated.
- +5 points more if every player defeated at least one Horseman.
- +5 points per head defeated (excluding the Control head).
- +15 points more if all heads were defeated.
- +30 points per Round finished before the Round limit.
- +1 point per unflipped card in the Dummy Player's Deed deck.
- +5 points if "End of the Round" wasn't announced yet in your last Round.

### Solo

Base = your Fame. Standard Achievements Scoring, no Titles. Plus:
- +3 points per Horseman you defeated.
- +5 points per head you defeated (excluding the Control head).
- +15 points more if you defeated all heads.
- +30 points per Round finished before the Round limit.
- +1 point per unflipped card in the Dummy Player's Deed deck.
- +5 points if "End of the Round" wasn't announced yet in your last Round.

## Outcome (Won/Lost)

Won iff the Apocalypse Dragon was defeated; Lost otherwise. A score is always computed either way.
