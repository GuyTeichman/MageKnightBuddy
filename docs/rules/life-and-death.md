# Life and Death

Source: `Mage-Knight-Board-Game-Ultimate-Edition-Expansion-Rule-Books-September-2018.pdf` (Shades of Tezla expansion, Scenario Book), pages 22–23.

Unlike Volkare's Quest/Return, this scenario has a **purpose-built Solo variant** in the rulebook itself (not "any scenario can be played solo" via Solo Conquest's generic modifications). Not a v1 target (v1 is Solo Conquest only) — captured here as reference for when this scenario gets implemented.

## Overview

- Length: Six rounds (3 days and 3 nights).
- Purpose: the full scenario against the forces of life and death. The Elementalists and the Dark Crusaders are at war; the Council of the Void wants you to exploit that and fight into enemy territory to destroy the two Avatars of Tezla — Tezla's Spirit (Elementalist) and Dark Tezla (Dark Crusader).

### Special rules (apply to all variants)

- **Enemy tokens**: green, brown, and red enemy tokens are sorted by faction. Rampaging enemies are drawn from the Dark Crusader pile on northern map tiles, the Elementalist pile on southern map tiles, or a random faction on tiles in between (per a north/south tile-position diagram on p.22).
- **Magical Glades**: revealed on a southern Countryside tile → a green + brown Elementalist enemy token face up on it. Revealed on a northern Countryside tile → a Graveyard tile placed over it, then a green + brown Dark Crusader enemy token face up on the Graveyard tile. Revealed on a tile in between → an enemy token from a random faction (either an Elementalist token, or a Graveyard tile + Dark Crusader token).
- **Liberate Location**: entering a Magical Glade/Graveyard's space is an Action; fight both enemies there. Defeating only one leaves the rest on the space, attackable later. Defeating both liberates the location — gain an Artifact (Magical Glade) or a Spell (Graveyard), mark it with your Shield token, +1 Reputation. Night rules apply in a Graveyard/Necropolis, but none of the other Magical Glade/Hidden Valley/Graveyard/Necropolis effects work until liberated.
- **Cities**: the Green City tile can't be placed as a northern map tile; the Red City tile can't be placed as a southern map tile (redraw the next core tile instead and reshuffle the city tile back in). Placing the Green City tile also places the Hidden Valley location tile over its city space; placing the Red City tile also places the Necropolis location tile over its city space.
- **Faction Leader Locations**: the Elementalist faction leader is in the Hidden Valley; the Dark Crusader faction leader is in the Necropolis. Placing either location also places that faction's enemy tokens, face up, per the leader's level. Both locations must be entered (they aren't safe spaces) to attack the leader and followers there.

## Competitive

- Players: 2 to 4. Type: Competitive.
- Map Shape: Wedge (2–3 players) or Open limited to 4 columns (4 players).
- Countryside tiles: 7, 9, or 11 (exactly 3/4/5 should contain a Magical Glade).
- Core city tiles: 3 — the Green City tile, the Red City tile, and one other (friendly: each player puts one Shield token on it, no-one is leader).
- Core non-city tiles: 1, 2, or 3 (one less than the number of players). Dark Crusader and Elementalist leader tiles both start at level = number of players + 2.

**Scenario end**: both faction leaders defeated → all players get one last turn (game ends immediately if the Round ends during this).

**Scoring**: apply standard Achievements Scoring. In addition, count each player's Shield tokens on a Magical Glade or the Elementalist leader token — the most gets +5 Fame ("Greatest Enemy" of that faction). Separately, count Shield tokens on a Graveyard or the Dark Crusader leader token — the most gets +5 Fame. Ties broken in favor of whoever's Shield token sits on the highest level of the faction leader token.

## Cooperative

- Players: 2 to 4. Type: Cooperative.
- Map Shape: Fully Open.
- Countryside tiles: 7, 9, or 11 (exactly 3/4/5 should contain a Magical Glade).
- Core city tiles: 3, same as Competitive (Green City, Red City, one friendly other).
- Core non-city tiles: 2, 3, or 4 (equal to the number of players). Leader tiles both start at level = 2 × number of players + 4.
- Dummy Player: one standard Dummy Player.
- Cards/Skills: remove the four competitive Spells (109–112); use the Lost Legion expansion's cooperative interactive Skill tokens instead of competitive ones; remove competitive Skills from every player's Skill deck, including the Dummy Player's.
- Additional rules: Dummy Player takes a random Tactic first, then the real players choose theirs. Team Rules apply, except everyone is one team.

**Scenario end**: both faction leaders defeated → all players (except the Dummy Player) get one last turn (game ends immediately if the Round ends during this).

**Scoring**: one team score. Base = the lowest Fame among all players. Apply standard Achievements Scoring, except each category only scores the single player with the highest value (most negative for Greatest Beating) — no Titles. Then:
- +10 points per Avatar defeated, +15 more if both were defeated.
- +10 points if every player has a Shield token on at least one of the two faction leader tokens, +10 more if every player has one on *both*.
- +30 points for each Round finished before the Round limit.
- +1 point for each card remaining in the Dummy Player's Deed deck not yet flipped this Round.
- +5 points if "End of the Round" was not yet announced this Round.

## Solo

- Players: 1. Type: Solo.
- Map Shape: Wedge.
- Countryside tiles: 6 (exactly 3 should contain a Magical Glade).
- Core city tiles: 3 — Green City, Red City, one other friendly (both the player and the Dummy Player put a Shield token on it, no-one is leader).
- Core non-city tiles: 2 (the Dark Crusader and Elementalist leader tiles), both starting at level 6.
- Dummy Player: one standard Dummy Player.
- Cards/Skills: same as Cooperative (remove Spells 109–112; use Lost Legion cooperative interactive Skill tokens; remove competitive Skills from both decks).
- Additional rules: the player always chooses Tactics first, the Dummy Player takes one random card from those remaining. At the end of each Day/Night, remove both used Tactic cards from the game — each Tactic card is picked exactly once during the whole scenario (same mechanic as Solo Conquest — see `solo-conquest.md`).

**Scenario end**: both faction leaders defeated → the player has one last turn (the Dummy Player does not).

**Scoring**: base = your Fame. Apply standard Achievements Scoring, no Titles. Then:
- +10 points per Avatar defeated, +15 more if both were defeated.
- +30 points for each Round finished before the Round limit.
- +1 point for each card remaining in the Dummy Player's Deed deck not yet flipped this Round.
- +5 points if "End of the Round" was not yet announced this Round.

## Outcome (Won/Lost) — all variants

Won iff both Avatars (both faction leaders) were defeated; Lost otherwise. A score is always counted either way — the rulebook is explicit that failing still lets you "count your score to see how good you were."
