# Mines Liberation

Source: `Mage-Knight-Board-Game-Ultimate-Edition-Rule-Book-September-2018.pdf` (base rulebook, Scenario Book), page 20.

**Competitive only** — the rulebook prints no Solo or Cooperative variant for this scenario, unlike e.g. First Reconnaissance or the Apocalypse Dragon scenarios. Not a v1 target (v1 is Solo Conquest only) — captured here as reference for when this scenario gets implemented (whether as-is for multiplayer, or house-ruled into a solo variant).

## Overview

- Players: 2 to 4. Type: Competitive.
- Length: Four rounds (2 days and 2 nights).
- Purpose: a shorter scenario similar to the standard one (Full Conquest), but with different goals — mines in this friendly kingdom are occupied by enemies, and without a supply of crystals, Mages cannot serve the land.

## Setup

- Map Shape: Wedge (2 or 3 players), or Open limited to 4 columns (4 players).
- Countryside tiles: 8, 9, or 11 (remove only tiles with no mines on them).
- Core city tiles: 1 (always the Red city).
- Core non-city tiles: 1, 2, or 3 (the one without a mine is always amongst the removed ones).
- Cities: the city is friendly. Each player puts one Shield token on it, but no-one is its leader.

## Special rules

- **When a mine is revealed**: put one green (if on a Countryside tile) or red (if on a Core tile) enemy token on it, face up. Also add one brown enemy token, face down (one enemy is hidden in the deeps). Arrange them so the reverse of the brown token is visible too, to distinguish it from rampaging enemies.
- **Liberating a mine**: enter it from its space (as an Action) and fight both enemies. Night rules apply as in a Dungeon (but you can use your Units). If you don't defeat both, the remaining token(s) stay on the space — they can be attacked again later.
- Whoever defeats the last enemy liberates the mine. They mark it and gain +1 Reputation (+2 if the mine is on a Core tile).
- Until liberated, mines produce no crystals. Once liberated, they produce as usual (production starts immediately, so the conqueror gets one crystal on the turn they liberated the mine).
- At the start of each Day or Night, each player gains one crystal from every mine they've already conquered, no matter how far away it is on the map.
- Note: if there aren't enough brown tokens, use a substitute for brown monsters that are on the map but not yet revealed; draw a new brown monster when it's about to be revealed.

## Scenario end

When all map tiles are revealed and all mines are liberated, all players (including whoever liberated the last mine) have one last turn. If the Round ends during this, the game ends immediately.

## Scoring

If you liberated all mines, the mission was successful. In any case, apply standard Achievements scoring. In addition:

- A player scores 4 Fame for each of their mines in Countryside, and 7 Fame for each mine on a Core tile.
- Whoever scores the most Fame this way gets an additional +5 Fame as the Greatest Liberator.

## Outcome (Won/Lost)

Won iff all mines were liberated; Lost otherwise. A score is always computed either way.
